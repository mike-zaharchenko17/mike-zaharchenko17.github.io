    InfScraper

    GUI Instagram Scraper for Marketers

    View on

    Description

    InfScraper is a GUI-enabled Instagram scraping tool intended to help digital marketers quickly and easily find and identify influencers. In more technical terms, InfScraper retrieves public profile data from Instagram and performs a DFS on the tree of related users to find similar profiles. With this, it's able to leverage Instagram's powerful recommendation algorithm to make a digital marketer's work simpler. It is heavily asynchronous to ensure high performance even under heavy load.

    Methods/Technologies Used

    Java, JavaFX, MongoDB Atlas, GSON
    Concurrency/Multithreading, CompletableFuture, Maven

    Development Achievements

    • Developed an architecture based on MVT and leaning heavily into the producer-consumer pattern
    • Built a rich user interface using JavaFX.
    • Used Java’s concurrency libraries to perform concurrent queries and API calls, vastly improving application performance
    • Stored/queried results in a MongoDB Atlas cluster and used a driver to interface with Mongo

    Doodlebug the Game

    Simple Terminal-Based Game in C++

    View on

    Description

    Doodlebug the Game is an independent project that I worked on while completing the Tandon Bridge program. Bridge is an intensive 24-week program meant to build non-STEM students' prerequisites to pursue graduate studies in computer science. Doodlebug is an infamous assignment from the program that is given to every cohort, where students are asked to simulate a small ecosystem where doodlebugs need to eat ants to survive and the two species reproduce at different rates. It is meant to be many students' first exposure to building (relatively) large software systems with lots of moving parts. After finishing the initial project, I took it a step further and built an ascii-based terminal game inspired by the assignment. I shared it with my cohort to help everyone blow off some steam ahead of our final exams.

    Methods/Technologies Used

    C++, OOP, Event-Driven Programming

    Development Achievements

    • Refined an existing idea and made it interactive
    • Implemented event-driven programming in C++ and created a turn-based system with an addictive gameplay loop
